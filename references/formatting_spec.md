# Chinese Academic Paper Formatting Specification

## Page Layout

- Top/Bottom margins: 2.54 cm
- Left/Right margins: 3.17 cm
- Line spacing: 1.5x throughout (except ToC which may adjust)
- Character spacing: standard, no scaling

## Page Numbering

- Front matter (Chinese/English title, abstract, keywords): **Roman numerals** (i, ii, iii...)
- Body (sections, references, appendices, acknowledgements): **Arabic numerals** (1, 2, 3...)
- Position: footer, centered, 六号 (7.5pt), 宋体

## Font Size Reference

| Name | Point Size |
|------|-----------|
| 三号 | 16pt |
| 四号 | 14pt |
| 小四 | 12pt |
| 五号 | 10.5pt |
| 六号 | 7.5pt |

## Title (Chinese)

- Font: 黑体 (SimHei), 三号 (16pt), bold, centered
- Maximum 20 characters
- No uncommon English abbreviations

## Title (English)

- Font: Times New Roman, 三号 (16pt), bold, centered

## Chinese Abstract

- Label "摘要": 黑体, 小四 (12pt), bold, left-aligned
- Body: 宋体 (SimSun), 小四 (12pt)
- ~200 characters
- Third person (no 本文, 论文)
- Innovation points labeled inline in 黑体 bold
- No mathematical formulas, figures, tables, or reference numbers

## Chinese Keywords

- Label "关键词": 黑体, 小四 (12pt), bold, left-aligned
- Keywords: 宋体, 小四 (12pt), 3-8 items, semicolon-separated

## English Abstract

- Label "Abstract": Times New Roman, 小四 (12pt), bold
- Body: Times New Roman, 小四 (12pt)
- Innovation points in *italics* (not bold)
- Content matches Chinese abstract

## English Keywords

- Label "Key words": Times New Roman, 小四 (12pt), bold
- Keywords: Times New Roman, 小四 (12pt), lowercase (except acronyms), semicolon-separated

## Table of Contents

- Title "目录": 三号, 黑体, bold, centered, 2-character spacing between characters
- Level 1 entries: 小四, 黑体, bold, left-aligned
- Level 2 entries: 小四, 宋体, indent 2 characters
- Level 3 entries: 小四, 宋体, indent 2 characters
- Page numbers right-aligned with dot leaders

## Section Headings (Body)

### Level 1 Heading (一级标题)
- Font: 四号 (14pt), 黑体, bold, left-aligned
- Numbered: "1", "2", "3"...

### Level 2 Heading (二级标题)
- Font: 小四 (12pt), 黑体, bold, left-aligned
- Numbered: "1.1", "1.2", "2.1"...

### Level 3 Heading (三级标题)
- Font: 小四 (12pt), 宋体, bold, left-aligned
- Numbered: "1.1.1", "1.1.2"...
- No level 4 headings allowed

## Body Text

- Font: 小四 (12pt), 宋体
- Numbers and English: Times New Roman
- Line spacing: 1.5x
- Character spacing: standard, no scaling
- Reference citations: superscript [1] format, Times New Roman, 小四

## Tables (三线表)

- Caption: 小四, 宋体, bold, centered, "表1.1 表的名称"
- Table number: two-level (section.table_number)
- Header row: 小四, 宋体, centered
- Data cells: 小四, 宋体, centered
- Three-line format: top thick border, header-bottom thin border, table-bottom thick border
- Notes below table: 五号 (10.5pt), 宋体
- Note labels (注): 五号, 宋体, bold
- Multiple notes use ①, ②... markers

## Figures

- Caption: 小四, 宋体, bold, centered, "图1 图的名称"
- Figure number: single-level sequential
- Below the figure

## Equations

- Equation centered on line
- Equation number right-aligned: "(1)", "(2)"...
- Variables: Times New Roman, 小四

## References

- Title "参考文献": 五号 (10.5pt), 黑体, bold, centered
- No section number prefix
- Content: 五号 (10.5pt), 宋体
- Reference numbers in brackets: [1], [2]...

### Reference Formats

#### 1) Journal Article [J]
[序号] 主要责任者.文献题名[J].刊名,年,卷(期):起止页码.

#### 2) Book/Monograph [M]
[序号] 主要责任者.书名[M].其他责任者(选择项).版本(第1版不标注).出版地:出版者,出版年:页码(选择项).

#### 3) Conference/Collected Papers [C]/[G]
[序号] 主要责任者.析出文献题名[文献类型标识] // 编者.原文献名.出版地:出版者,出版年:析出文献起止页码.

#### 4) Dissertation [D]
[序号] 作者.题名:[D].保存地:保存者,年份.

#### 5) Newspaper [N]
[序号] 主要责任者.文献题名[N].报纸名,出版日期(版次).

#### 6) Electronic Resource [EB/OL]
[序号] 主要责任者.电子文献题名[EB/OL].电子文献的出处或可获得地址,发表或更新日期/引用日期(任选).

#### 7) Patent [P]
[序号] 专利申请者.专利题名[P].专利国别,专利号,出版日期.

#### 8) Technical Standard [S]
[序号] 起草责任者.标准代号 标准顺序号—发布年 标准名称[S].出版地:出版者,出版年.

#### 9) Other/Undefined [Z]
[序号] 主要责任者.文献题名[Z].出版地:出版者,出版年.

#### 10) Foreign Language References
Same format as Chinese. Title words capitalized. Add ISSN (SXXXX-XXXX) after journal name.
Example: [1] KANAMORI H. Shaking without Quaking [J]. Science (S0036-8075), 1998, 279: 2063.

## Appendices

- Title: no section number prefix
- Example: "附录1", "附录2"
- Content formatting same as body text

## Acknowledgements (致谢)

- Title: no section number prefix
- Centered or left-aligned
- Content formatting same as body text
